MoreMenu
========

A MediaWiki gadget to add dropdown menus with links to common tasks,
user and page analytic tools and logs.

User documentation: https://meta.wikimedia.org/wiki/MoreMenu

## Contributing

* `npm install` to install dependencies
* `npm run build` to compile the source [MoreMenu.js](https://github.com/MusikAnimal/MoreMenu/blob/master/MoreMenu.js)
  to [MoreMenu.dist.js](https://github.com/MusikAnimal/MoreMenu/blob/master/MoreMenu.dist.js)

MoreMenu.dist.js is what should be used on the wiki.
Refer to https://meta.wikimedia.org/wiki/MoreMenu#Setup for on-wiki installation.
